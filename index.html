PDS Registry

PDS Registry provides functionality for tracking, searching, auditing, locating, and maintaining artifacts within the system. These artifacts can range from data files and label files, schemas, dictionary definitions for objects and elements, services, etc…

The PDS Registry is both the Service which provides the functions listed above overall the different nodes of the PDS in an integrated way, and it is the software Application which implements these functions at the node level.

This documentation is intended for the users and administrators who contribute to the PDS Registry Service. The users of the PDS Registry Service through the web API should refer to the PDS Search API user’s guide
